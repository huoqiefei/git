### Step 1 准备工作 本地安装git bash: https://gitforwindows.org/ 注册github帐号： https://github.com/ ### Step 2 git使用 在本地项目目录中初始化git: ``` git init ``` 拉取远程仓库数据: ``` git clone https://github.com/username/projectname.git ``` 设置用户名及密码： ``` git config global user.name "username" git config global user.email "username@domain.com" ``` 查看本地仓库状态: ``` git status ``` 添加文件： ``` git add filename.file ``` 添加远程仓库： ``` git remote add origin https://github.com/username/projectname.git ``` 提交代码到master分支 ``` git push origin master ```
